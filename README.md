# Chatbot

A simple frontend chatbot that simulates real-time conversations using HTML, CSS, and JavaScript.

# Smart Chatbot

## Description
This project is a simple and interactive chatbot built using HTML, CSS, and JavaScript. It provides a clean user interface and simulates real-time conversation using predefined responses and basic logic.

---

## Features
- Interactive chat interface  
- Real-time message display  
- Clean and responsive design  
- Loading/typing animation  
- User avatar support  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

/chatbot-project

---

## How It Works
1. User enters a message in the input field  
2. The message is displayed in the chat interface  
3. JavaScript processes the input  
4. A loading animation is shown  
5. The chatbot replies with a predefined response  

---

## How to Run
1. Download or clone this repository  
2. Open the project folder  
3. Open `index.html` in your browser
